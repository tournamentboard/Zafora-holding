Landing Page Management Admin Tool Requirements
1. Main Purpose

The admin tool should allow approved admins to update the full Zafora Holding landing page from one dashboard, including:

Hero section
Stats and numbers
Services
Pillars
Project pipeline
Countries and sectors
Methodology steps
Leadership/about content
Images and icons
Buttons and links
Footer information
Contact details
SEO content

The goal is simple: no developer needed for normal content updates.

2. Admin Roles
Super Admin

Full control over everything.

Can:

Edit all landing page content
Upload images
Publish updates live
Approve drafts
Manage admins
View change history
Restore previous versions
Control contact info
Control SEO
Control footer
Manage project pipeline

Content Admin

Can update regular content but cannot publish without approval.

Can:

Edit text
Upload images
Create draft updates
Update services
Update sectors
Update methodology
Update project cards

Cannot:

Publish live changes
Delete major sections
Change admin users
Edit billing or system settings

Reviewer / Approver

Can review and approve changes.

Can:

Preview landing page changes
Approve or reject drafts
Leave comments
Publish approved updates

Viewer

Read-only access.

Can:

View content
Preview page
Review current live version

10. Security Requirements

The admin tool should include:

Secure login
Role-based access control
Two-factor authentication

Session timeout
Admin activity log
Change approval workflow
No public access to admin pages

Finance or business-sensitive controls should only be available to Super Admin.

14. Final Requirement Statement

Zafora Holding needs a simple content management admin tool that allows authorized admins to update the full landing page, manage project pipeline content, edit services, replace images, control CTA buttons, update company information, manage SEO, preview changes, and publish approved updates without touching code. The system must include role-based permissions, draft approval, version history, image management, and secure admin access.
